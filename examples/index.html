    Examples¶

    Required properties

    The project name, specVersion and bomFormat are required. All others are optional.

    Minimum example¶

    {
  "specVersion": "1.0",
  "bomFormat": "Naikan",
  "project": {
    "name": "Naikan III"
 }
}
