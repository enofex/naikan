# Examples

!!! note "Required properties"

     The **project name**, **specVersion** and **bomFormat** are required. All others are optional.

## Minimum example

``` json
{
  "specVersion": "1.0",
  "bomFormat": "Naikan",
  "project": {
    "name": "Naikan III"
 }
}
```
